Продукты

Клиенский терминал
Используется клиентами Банка для совершения торговых операций с Банком
Дилерский терминал
Используется дилингом Банка для торгов с поставщиками ликвидности от имени Банка
Терминал менеджера
Используется администраторами системы для её управления
Терминал сейлза
Позволяет организовать торговлю через посредника - сейлза

Особенности системы

Импорт и экспорт котировок
Котируйте по заданным в Excel формулам. Загружайте или выгружайте котировки с помощью Excel
Синтетическая ликвидность
Оцените широкие возможности для настройки кросс-курсов и расчетных котировок
Цифровая подпись
Обеспечит безопасность торговых операций клиентов
Алгоритмы перекрытия
Уменьшите влияния рынка (market impact) при торговле большими объемами
Передача отчетности
Отправляйте информацию по сделкам в бэк-офис системы банка в режиме реального времени
Лимитная торговля
Настраивайте SPOT и Forward лимиты и критерии лимитирования. Система интегрирована с банковской базой данных
